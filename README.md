# 🎓 SecretCoder - E-Learning Website

SecretCoder is a responsive E-Learning web application developed using *PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap*. The platform allows users to create an account, enroll in courses, make payments, manage purchases, and update their profiles through a simple and user-friendly interface.

---

## 🚀 Features

- 🔐 User Registration (Sign Up)
- 🔑 Secure Login
- 🔒 Forgot Password
- 📩 OTP Verification
- 🔄 Reset Password
- 👤 User Dashboard
- 📚 Browse Courses
- 💳 Course Payment
- 🛒 My Purchases
- ✏️ Edit Profile
- ⭐ Course Review & Rating
- 📞 Contact Form
- 📱 Responsive Design

---

## 🛠️ Tech Stack

*Frontend*
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

*Backend*
- PHP

*Database*
- MySQL

*Server*
- XAMPP / Apache
